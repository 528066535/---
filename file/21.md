## 打印功能实现方案

正好在工作中有一个打印功能，早期打印直接用jquery实现，现在框架去掉jquery，所以手撸一份。

#### 基本实现方法

```
window.print()

document.execCommand('print')。
```

此方法打印的是当前页面，很多时候我们只要打印指定的页面。

这时候我们就应该用一些神奇的办法了

### 1. 使用media控制

a. 使用 @media print

@media print {
	.no-print {
    display: none
  }
}

b. 内联样式

```
<style type="text/css" media="print">
</style>
```

c. 使用@import

```
@import url("print.css") print;
```
